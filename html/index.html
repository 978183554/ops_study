🚀 云原生运维实战

基于 Docker Compose 的多容器编排项目

📱 前端 (Nginx) ✓ 运行中
⚙️ 后端 API (Go) ✓ 运行中
🗄️ 数据库 (MySQL) ✓ 运行中
💾 缓存 (Redis) ✓ 运行中
🔗 API 主页 💓 健康检查 📊 Prometheus 指标 ⚙️ 配置信息

项目架构:

• 前端通过 Nginx (端口 80) 提供

• 后端 API 运行在 Go (端口 8080)

• MySQL 数据库 (端口 3306)

• Redis 缓存 (端口 6379)

• 所有服务通过 mynetwork 网络通信

📚 学习要点:

✓ 理解 Docker Compose 多容器编排

✓ 掌握容器间网络通信

✓ 学习环境变量配置管理

✓ 体验现代应用架构
